# Cisco Starter Repository

A collection of projects and tools developed as part of Cisco's backend Forage program.

## Projects

### SEXTANT
A real-time network diagnostics dashboard built with React.

**Features:**
- Public IP address detection (IPv4/IPv6)
- Real-time latency monitoring with WebSocket communication
- Live time-series chart visualization
- Responsive web interface

**Technologies:** React, Node.js, WebSocket, Chart.js

**Location:** `sextant/` directory

**Running SEXTANT:**
1. Navigate to `sextant/` directory
2. Install dependencies: `npm install`
3. Start WebSocket server: `node server.js` (port 55455)
4. Start React app: `npm start` (port 3000)
5. Open browser to `http://localhost:3000`
